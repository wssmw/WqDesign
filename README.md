# Vue 3 + Vite 项目

## 项目介绍
这是一个基于 Vue 3 和 Vite 的模板项目，旨在帮助开发者快速搭建 Vue 3 应用。

## 功能特性
- 使用 Vue 3 `<script setup>` 语法
- 集成 Vite 构建工具
- 支持 TypeScript

## 快速开始
### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产环境
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```
