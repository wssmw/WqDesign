# 什么是let,const,var

| 特性         | var                          | let                 | const               |
| ------------ | ---------------------------- | ------------------- | ------------------- |
| 作用域       | 函数作用域                   | 局部作用域          | 局部作用域          |
| 变量提升     | ✅ (提升并初始化为undefined) | ✅ (提升但不初始化) | ✅ (提升但不初始化) |
| 重复声明     | ✅ 允许                      | ❌ 不允许           | ❌ 不允许           |
| 初始值       | 可选                         | 可选                | 必须                |
| 重新赋值     | ✅ 允许                      | ✅ 允许             | ❌ 不允许           |
| 暂时性死区   | ❌ 无                        | ✅ 有               | ✅ 有               |
| 全局对象属性 | ✅ (全局var会成为window属性) | ❌                  | ❌                  |
