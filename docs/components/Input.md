# Input 输入框

<preview path="../demos/components/Input/Basic.vue" title="基础用法" description="Input 组件的基础用法"></preview>

## 介绍

用于输入文本内容的表单控件。

## API

| 参数        | 说明     | 类型    | 默认值 |
| ----------- | -------- | ------- | ------ |
| v-model     | 绑定值   | string  | -      |
| placeholder | 占位符   | string  | -      |
| disabled    | 是否禁用 | boolean | false  |
